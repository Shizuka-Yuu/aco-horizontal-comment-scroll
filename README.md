#  (v1.0.0)
ACO Horizontal Comment Scroll
AbemaTV（`https://abema.tv/*`）のライブ配信および番組において、公式のコメントデータを取得し、動画プレイヤー上に某動画サイト風にコメントを右から左へ流すブラウザ拡張機能（Chrome・Firefox用 MV3仕様）です。

本拡張機能は個人開発による非公式のソフトウェアです。
